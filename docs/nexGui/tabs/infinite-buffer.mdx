Infinite Buffer is a modal log viewer with search and navigation tools.

<!-- TODO: Add a screenshot of the Infinite Buffer modal. -->

## How to open

- Press `Ctrl+Shift+L`, or
- Press Page Up when the infinite display is enabled.

## Features

- Search through the buffered log.
- Pin scroll position and jump to results.
- Close with `Esc`.

## Notes

- Modal size and position are stored in `nexGuiSettings.modal`.
