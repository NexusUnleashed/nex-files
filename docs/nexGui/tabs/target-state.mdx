Target State visualizes limb damage and trauma on your current target.

<!-- TODO: Add a screenshot of the Target State tab. -->

## What it shows

- Limb damage percentages with a body silhouette.
- Trauma and limb damage overlays based on afflictions.

## Dependencies

- Requires the `insight` package for target tracking data.

## Notes

- The limb display uses images hosted in the nexGui package CDN.
