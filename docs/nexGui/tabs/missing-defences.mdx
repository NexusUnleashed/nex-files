Missing Defences lists defences that are expected but not currently active.

<!-- TODO: Add a screenshot of the Missing Defences tab. -->

## What it shows

- `nexGui.defences` list compared against `GMCP.Defences`.
- Entries rendered with the Defence component.

## Notes

- The defence list is typically populated by class customizations.
- This tab is most useful alongside nexSys.
