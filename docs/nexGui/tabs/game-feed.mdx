Game Feed shows the Achaea public gamefeed stream inside a nexGui tab.

<!-- TODO: Add a screenshot of the Game Feed tab. -->

## What it shows

- Live gamefeed entries (deaths, level gains, etc.).
- Timestamped entries with colored segments.

## Data source

- `https://api.achaea.com/gamefeed.json` (polled every 5 seconds).

## Notes

- The feed retains the most recent 100 entries.
