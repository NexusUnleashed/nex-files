title: System Stream
---

System Stream shows affliction and defence changes with timestamps.

<!-- TODO: Add a screenshot of the System Stream tab. -->

## What it shows

- `+aff` and `-aff` entries from GMCP affliction events.
- `+def` and `-def` entries from GMCP defence events.
- Each entry includes a timestamp and styled label.

## Notes

- Enable **System Stream** in Game Options to populate this tab.
