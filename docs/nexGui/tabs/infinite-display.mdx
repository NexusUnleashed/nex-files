title: Infinite Display
---

Infinite Display replaces the Nexus output panel with a virtualized buffer.

<!-- TODO: Add a screenshot of the Infinite Display in use. -->

## What it does

- Renders output using a virtual list for smoother scrolling.
- Syncs with the virtual buffer and log export tools.

## How to enable

- Open **Game Options** > **Experimental**.
- Enable **Override Nexus Display**.

## Notes

- This feature sets `nexusclient.ui().layout().nexoverrides.output_main`.
- If output goes blank, disable and re-enable the toggle.
