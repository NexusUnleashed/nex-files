title: Game Options
---

Game Options is the primary configuration UI for nexGui. It includes three sub-tabs: Config, Style, and Experimental.

<!-- TODO: Add a screenshot of the Game Options tab (Config view). -->

## Config

- Nexus settings toggles (echo input, timestamps, prompts, tab strip).
- nexGui toggles for System Stream, Combat Stream, Msg Replace, and PVE damage numbers.

## Style

- Background image selection.
- Opacity and hue adjustments.
- Font size control for nexGui tabs.
- Ally/enemy and target formatting toggles.

## Experimental

- Login tracker (uses Achaea character API).
- Download on logout and virtualized logs.
- Override Nexus Display (infinite display renderer).
- Netflix mode sound alerts for communication.
