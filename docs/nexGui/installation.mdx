## Install in Nexus

1. Download the `.nxs` file.
2. Drag and drop the file into the **Reflexes** panel in Nexus.
3. Reload the client or relog if you do not see the nexGui tabs.

## Verify

- Run `ng load` or `nexGui.startUp()`.
- Confirm the nexGui tabs appear next to the Map tab.
- Run `nexGui.version` in the console.
