# nexGear

:::caution Labs preview

nexGear is an alpha package. Its interface, storage model, and installation
process may change without notice.

:::

nexGear explores a more visual way to manage equipment in Achaea's Nexus
client. It combines a searchable gear inventory with set assignment and
persistent local preferences.

## Current direction

- Sortable and filterable equipment tables
- Official and user-defined gear sets
- Multi-set assignment for trinkets
- Local persistence between Nexus sessions
- Quick equip and item-management actions

## Documentation status

The package does not yet have a stable installation guide or public API
contract. Those pages will be added when the package is ready to move out of
Labs.

Until then, treat this page as a statement of direction rather than a support
guarantee.
