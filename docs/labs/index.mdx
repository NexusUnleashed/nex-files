# Labs

Labs is where promising nex* experiments live before their public interfaces
and documentation are ready for the main package catalog.

:::caution Experimental software

Labs projects may be incomplete, change without notice, or require manual
configuration. Treat them as previews rather than supported releases.

:::

## nexGear

An experimental equipment and gear-set manager with searchable inventory,
custom set assignment, and persisted selections.

[View the nexGear preview](./nexgear.mdx)

## nexWorld

An early exploration without a stable public documentation surface yet. It
will receive a dedicated page when its scope and installation path are ready.
