## Install in Nexus

1. Download the `.nxs` file.
2. Drag and drop the file into the **Reflexes** panel in Nexus.
3. Reload the client or relog if you do not see the config dialog.

## Verify

- In the developer console, run: `nexSys.version`.
- Use the config dialog to set your command separator and confirm settings are saved.
