# Build your Nexus setup with nex\*

nex\* is a family of focused packages for Achaea's Nexus client. Version 4
rebuilds the core experience around explicit state, predictable events, and
small public APIs that packages can safely share.

## Start here

- [Getting started](./getting-started.mdx) — understand the runtime and install your first package.
- [Browse packages](./packages.mdx) — compare supported packages and their dependencies.
- [How the ecosystem fits together](./ecosystem.mdx) — learn the shared runtime model.

## Current package families

| Area                | Packages                            | Purpose                                             |
| ------------------- | ----------------------------------- | --------------------------------------------------- |
| Core experience     | nexSys4, nexMap4, nexGui4, nexBash4 | Character state, navigation, interface, and hunting |
| Combat intelligence | insight                             | Target state and probabilistic combat tracking      |
| Foundations         | eventStream, nexAction              | Events, aliases, and trigger handling               |
