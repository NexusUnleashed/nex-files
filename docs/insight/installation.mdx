## Install in Nexus

1. Download the `.nxs` file.
2. Drag and drop the file into the **Reflexes** panel in Nexus.
3. Reload the client or relog if you do not see insight events.

## Verify

- Run `insight.version` in the console.
- Run `insight.currentAffs()` after setting a target.
